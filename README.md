# Flashing-Led-with-Arduino
Timed blinking LED project using an Arduino Mega 2560

Components Used (PC for uploading code to Arduino)
- 1x Arduino Mega 2560
- 1x LED
- 1x 220ohm Resistor
- 2x Jumper Wires
